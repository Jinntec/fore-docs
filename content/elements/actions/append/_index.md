### Description

Appends an entry to an `fx-repeat`.

> Note: this is deprecated. Please use `fx-insert`.

### Attributes

| Name | Description |
|------|-------------|
| ref | the XPath ref to append to |
| repeat | the `fx-repeat` to append to |
