## Description

Triggers immediate refresh.

## Attributes

| Name | Description                                                    |
|------|----------------------------------------------------------------|
| control | id of an control to be refreshed explicitly                    |
| force | marker attribute to force a full refresh - overrules `control` |
| self | searches upwards for `fx-control` and refreshes it |

## Events

none
