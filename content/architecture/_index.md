# Architecture

This chapter gives some insights into the inner workings of Fore.

![MVC](/images/mvc.svg)

Fore is build as a classical MVC architecture.

* **M**odel -> `<fx-model>` Element
* **V**iew -> Fore UI Elements
* **C**ontroller -> Fore Action Elements
